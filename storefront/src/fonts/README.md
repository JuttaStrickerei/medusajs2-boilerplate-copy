# Georgia Font für Frontend

Dieses Verzeichnis enthält die Base64-kodierten Georgia-Font-Dateien für die Frontend-Anwendung.

## Verwendung

Die Fonts können für Web-Fonts oder andere Frontend-Zwecke verwendet werden.

## Struktur

Die Datei `georgia-base64.ts` sollte Base64-kodierte Font-Daten enthalten, die dann im Frontend verwendet werden können.
